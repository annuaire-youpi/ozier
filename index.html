Suivi annuel de marge
Exercice —
💡 Comprendre votre marge
• CA = somme (tonnages livrés × tarifs clients) + (journées × tarif journée). Seules les commandes réellement effectuées comptent.
• Dépenses = entretien + réparation + pneus + carburant (déclaré par les chauffeurs) + autres frais véhicules
• Marge = CA − Dépenses. Le % de marge = Marge / CA — au-dessus de 15% c'est sain pour le BTP transport
💡 Si la marge décroche un mois, regardez d'abord la page 📊 Rapports → 🚛 Flotte : un camion en surconsommation tire la moyenne vers le bas.

Remontées issues des fiches journalières
💡 D'où viennent ces observations ?

Cette page centralise toutes ces remontées pour vous permettre d'anticiper les pannes plutôt que de les subir.
💡 Workflow recommandé : consultez cette page tous les matins. Si une observation est critique, contactez directement le chauffeur avant qu'il ne reparte.
